# GreenleafAudit — Smart App Download Redirect 📱

One smart link that automatically detects device type and redirects to the right app store.

## Features

✅ **Automatic Device Detection** — iOS → App Store | Android → Google Play
✅ **Beautiful Fallback Page** — Desktop users see both options
✅ **QR Code Ready** — Generate a QR for marketing materials
✅ **Deployed on Vercel** — Fast, global CDN, auto-scaling
✅ **No Dependencies** — Pure JavaScript, ~100 lines of code

## 🚀 Getting Started

See [DEPLOYMENT.md](DEPLOYMENT.md) for full setup instructions.

**TL;DR:** Push to GitHub → Deploy on Vercel → Done!
